# Shoregate Consulting Landing Page

This is a modern, responsive landing page for Shoregate Consulting, designed to showcase AI-powered solutions for small and medium businesses (SMBs).

## Features
  - AI Readiness Assessments
  - Quick-Win AI Pilots
  - AI Strategy Workshops
## How to Use
1. **Open `index.html`** in your browser to view the landing page.
2. **Customize Content:** Edit text, images, and links in `index.html` as needed for your business.
3. **Update Images:** Replace image files in the project directory to match your branding.
4. **Contact Form:** Submissions are sent via Formspree. Update the form action URL if you want to use a different service.
This site uses Google Analytics for traffic analysis. The tracking code is included in the `<head>` section of `index.html`. To use your own Google Analytics property, replace the tracking ID (`G-DE88TRRM15`) in the script tag with your own.

## File Structure
- `index.html` — Main webpage
- `alt_index.html` — Alternate design (optional)
- `AI_photo.jpg`, `ai_pilot.png`, `business_meeting_modern_office.jpg`, `consultants_around_table.jpg`, `Shoregate_Consulting_Logo.PNG` — Images used in the page

## Customization
- **Colors & Styles:** Adjust CSS in the `<style>` block of `index.html` for branding.
- **Navigation:** Add or remove sections as needed.
- **Integrations:** Replace Formspree and Calendly links with your own services if desired.
